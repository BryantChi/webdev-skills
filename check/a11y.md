# ♿ 無障礙檢查清單（WCAG 2.2 AA）

> 詳細規則見 [`../test/a11y-axe.md`](../test/a11y-axe.md) 與 [`../ui/a11y.md`](../ui/a11y.md)

## 自動化（必跑）

```bash
npx @axe-core/cli http://localhost:3000 --tags wcag2a,wcag2aa,wcag22aa
```

或在 Playwright E2E 內 `@axe-core/playwright`。

## 結構（CRITICAL）

- [ ] `<html lang="zh-Hant-TW">`（或對應語）
- [ ] 每頁唯一 `<h1>`
- [ ] 標題層級不跳階（h1 → h2 → h3）
- [ ] 使用 landmark：`<header>` `<main>` `<nav>` `<footer>` `<aside>`
- [ ] Skip-to-content 連結（首個焦點）

## 圖片與媒體

- [ ] 內容圖有 `alt`（描述功能）
- [ ] 裝飾圖 `alt=""`（不是省略）
- [ ] 影片有字幕（CC）
- [ ] 自動播放預設靜音或不自動播

## 表單

- [ ] 每個 input 有 `<label>` 關聯（`for` / wrapping）
- [ ] 錯誤訊息用 `aria-describedby`
- [ ] required 欄位有 `aria-required` 或 visual marker
- [ ] 表單送出後焦點跳到結果或錯誤
- [ ] 不只用顏色表達錯誤（加 icon / 文字）

## 互動

- [ ] 互動元素是 `<button>` 或 `<a>`，不是 `<div onClick>`
- [ ] 鍵盤可達（Tab 順序合邏輯）
- [ ] `:focus-visible` 顯著輪廓（≥2px、對比足）
- [ ] Modal 開啟焦點移入、關閉移回
- [ ] Modal Esc 可關
- [ ] Dropdown 方向鍵可導

## 顏色與對比

- [ ] 文字對比 ≥ 4.5:1（一般）/ 3:1（≥18pt 或粗 14pt）
- [ ] UI 元件對比 ≥ 3:1（icon、focus ring、border）
- [ ] 不只用顏色傳達資訊（加 icon / 文字）
- [ ] 支援 `prefers-color-scheme`（dark mode）

## 動畫

- [ ] 支援 `@media (prefers-reduced-motion)` 關動畫
- [ ] 自動播放動畫 ≤ 5 秒，或提供暫停
- [ ] 不用快速閃爍（>3 次/秒，避免癲癇觸發）

## 觸控目標

- [ ] 觸控目標 ≥ 44×44px（WCAG 2.2 新增）
- [ ] 點擊間距 ≥ 8px

## 螢幕閱讀器測試（人工）

| 平台 | 工具 |
|------|------|
| macOS | VoiceOver（Cmd+F5） |
| Windows | NVDA（免費） |
| iOS | VoiceOver |
| Android | TalkBack |

每次 release：用一個閱讀器跑核心流程（登入、結帳、表單）。

## 焦點順序測試

純鍵盤走過頁面：
- [ ] Tab 順序合邏輯
- [ ] 沒有「焦點消失」（被 hidden 元素吸住）
- [ ] Modal 焦點被困在內（focus trap）
- [ ] 跳過導航有效

## 對應 WCAG 條目（速查）

| 條目 | 條號 | 自動可驗 |
|------|------|---------|
| 圖片 alt | 1.1.1 | ✅ |
| 顏色對比 | 1.4.3 | ✅ |
| 鍵盤可達 | 2.1.1 | 部分 |
| 焦點可見 | 2.4.7 | ✅ |
| 標題與標籤 | 2.4.6 | 部分 |
| Reduce motion | 2.3.3 | 部分 |
| 觸控目標 | 2.5.5 | ✅ |
